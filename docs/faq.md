# FAQ

Where are designs stored?
- In a .superdesign/ folder in your workspace.

Does it work with Cursor/Windsurf/Claude Code?
- Yes. You can copy prompts into these IDEs and also select Claude Code as a provider.

Can I use a local OpenAI-compatible server?
- Yes. Select OpenAI in custom agent settings, set any API key, and specify superdesign.openaiUrl (e.g., http://127.0.0.1:1234/v1 for LM Studio). Note that primary provider selection is managed via superdesign.llmProvider.

How do I reset the welcome screen or clear chat?
- Commands: Superdesign: Reset Welcome Screen; Superdesign: Clear Chat.

How do I open settings quickly?
- Command: Superdesign: Open Settings.
