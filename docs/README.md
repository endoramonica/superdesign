- Commands – commands, menus, activation, and views
- Architecture – code organization and main flows
- Webview – UI structure, canvas, chat, and CSP
- Providers – LLM provider abstraction, Claude API/Code
- Tools – built-in workspace tools (read, write, edit, grep, etc.)
- Development – workflows, scripts, linting, testing
- Troubleshooting – common issues and fixes
- FAQ – common questions
- Security – scopes, CSP, API keys
